# LanceKill-Game
A 2D cold weapon game

This game is suitable for PC. You will fight in a battlefeild. 

There are 2 gamemodes:
- 4vs4 Rounded
- 4vs4 Team Deadmatch

Controls:
- W : Move upward
- S : Move downward
- A : Move leftward
- D : Move rightward
- LShift : Sprint
- Mouse Left Click : Attack
- Mouse Right Click : Skill Attack
- Esc : Pause
